Introducción a la programación

Box-model

Es el modelo que describe cómo los elementos HTML son renderizados en la página, incluyendo márgenes, bordes, rellenos y el contenido en sí.

Border

Es el borde alrededor de un elemento HTML, se define con propiedades como border-width, border-style, y border-color.

Break

La etiqueta <br> en HTML se usa para insertar un salto de línea dentro de un párrafo o texto.

Camelcase

Es una convención de nomenclatura donde las palabras se escriben sin espacios, y la primera letra de cada palabra, excepto la primera, es mayúscula.

cd

Comando utilizado en la terminal para cambiar de directorio. No es específico de HTML, pero se usa en el contexto de trabajar con archivos HTML.

cd ../

Comando de la terminal que mueve al directorio superior en la estructura de archivos.

Clase

En HTML, las clases se definen con el atributo class para aplicar estilos CSS o para referirse a un grupo de elementos de manera común.

Clear

Propiedad de CSS que controla cómo los elementos flotantes (con float) afectan el flujo del documento.

code .

En la terminal, abre Visual Studio Code en el directorio actual. No está relacionado directamente con HTML.

Comandos

Son instrucciones dadas a la terminal o al navegador para realizar tareas específicas, como git commit o git push, aunque no son específicos de HTML.

cp

Comando de la terminal utilizado para copiar archivos o directorios.

Develop

Es una rama en Git donde se realizan los desarrollos o nuevas características antes de ser fusionadas a la rama principal (main).

Etiquetas

Las etiquetas en HTML (como <div>, <p>, <a>) definen la estructura del contenido de la página web.

Feauture

Se refiere a una nueva funcionalidad que se implementa en el desarrollo de software, generalmente gestionada en ramas de Git llamadas "feature".

Git Bash

Es una interfaz de línea de comandos que proporciona una emulación de Bash (un intérprete de comandos de Linux) para Windows, usada principalmente con Git.

git add .

Comando de Git que añade todos los cambios del directorio actual al área de preparación (staging area) para ser commiteados.

git commit -m

Comando de Git que guarda los cambios realizados en el repositorio, con un mensaje que describe las modificaciones.

GitHub

Plataforma de hospedaje para proyectos de programación. Utiliza Git como sistema de control de versiones. Permite colaborar con más desarrolladores.

git push origin main

Comando de Git que sube los cambios locales a la rama principal (main) de un repositorio remoto en GitHub.

git status

Comando de Git que muestra el estado actual de los archivos en el repositorio. Indica si estos han sido modificados, añadidos o están listos para ser commiteados.

HTML

Lenguaje de marcado utilizado para crear la estructura de las páginas web. HTML define elementos básicos como encabezados, párrafos y enlaces.

IDE

Entorno de Desarrollo Integrado (por sus siglas en inglés). Herramienta que da todo lo necesario para escribir, probar y depurar código, como VS Code.

IA

Inteligencia Artificial, campo de la informática que se ocupa de crear máquinas que realizan tareas que normalmente requieren inteligencia humana.

input

Elemento HTML que permite crear controles interactivos en formularios web, como campos de texto, casillas de verificación o botones.

Inspector Web

Herramienta de desarrollo que se encuentra en los navegadores web, que permite inspeccionar y modificar el HTML y CSS de una página web.

label

Etiqueta HTML usada para definir etiquetas de texto asociadas a controles de formulario, mejorando la accesibilidad y la usabilidad.

ls

Comando utilizado en la terminal para listar los archivos y directorios en el directorio actual.

Main

Es la rama principal en un repositorio Git, donde se encuentran los cambios más estables y listos para producción.

Margin

Propiedad CSS que establece el espacio alrededor de un elemento, separándolo de otros elementos en la página.

Markdown

Lenguaje de marcado ligero, sirve para escribir documentos de texto plano que se pueden convertir a HTML. Comúnmente usado en archivos README.

Memoria update

Actualización de la memoria en el contexto de software, donde los cambios en el código o los datos son almacenados en la memoria del sistema.

mKdir

Comando utilizado en la terminal para crear nuevos directorios (carpetas) en el sistema de archivos.

Padding

Propiedad CSS que define el espacio interior de un elemento, es decir, el espacio entre el contenido y el borde del elemento.

Pull request

En GitHub, un "pull request" es una solicitud para que los cambios hechos en una rama sean revisados e integrados a otra rama, generalmente a la rama principal.

pwd

Comando de la terminal que muestra el directorio actual en el que se encuentra el usuario.

Repositorio

Espacio donde se guarda y gestiona un proyecto en Git. Puede estar localmente en tu computadora o en un servidor como GitHub.

rm

Comando de la terminal utilizado para eliminar archivos o directorios del sistema.

Selectores

En CSS, los selectores permiten seleccionar y aplicar estilos a los elementos HTML según sus atributos, clases o identificadores.

snake_case

Convención de nomenclatura donde las palabras se escriben en minúsculas y separadas por guiones bajos (_), comúnmente usada en nombres de archivos.

Style

Etiqueta HTML que se usa para incluir reglas de CSS dentro de un documento HTML.permite definir estilos, como colores, fuentes, tamaños y alineaciones
